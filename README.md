# artinscience · gallery

A public gallery of plates from **artinscience** — data-driven generative vector
art, drawn by machine and finished by hand on paper.

Each piece is shown here as a digital preview (SVG/PDF). What's sold is the
**unique physical plate**: a one-of-one pen-plotted original. The files are free
to view; the hand-made object is the art.

## Volume I — Ohio: Ten Cities, 200 Years

Data-cartography as industrial art — Ohio cities told through maps and data
across ~1820s–2020s, drawn by machine and finished by hand. *Work in progress.*

See [`ohio/`](ohio/) for the current plates and teasers.
